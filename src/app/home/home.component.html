F1 Angular App

Aplicación web de Fórmula 1

Descripción del Proyecto

Aplicación web moderna desarrollada con Angular que consume la API pública de Fórmula 1 (f1api.dev). Permite explorar equipos, buscar pilotos, consultar clasificaciones y visualizar estadísticas de temporadas mediante gráficas interactivas.

Stack Técnico

Angular 20 • Standalone Components • NG-Zorro • RxJS • Chart.js
